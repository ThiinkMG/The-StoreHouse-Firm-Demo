# Collaborating on This Project

Welcome! This document provides information on how to collaborate on the **FAFSA Timeline Generator Tool**.

## Getting Started

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/ThiinkMG/FAFSA-Timeline-Generator-Tool.git
    ```
2.  **Open the Project**:
    You can open `index.html` in your browser to see the tool in action.
3.  **Making Changes**:
    - Create a new branch for your feature or fix: `git checkout -b feature/your-feature-name`
    - Commit your changes: `git commit -m "Add some feature"`
    - Push to the branch: `git push origin feature/your-feature-name`
    - Create a Pull Request on GitHub.

## Adding Collaborators

To add a friend to this project:
1.  Go to the [Repository Settings](https://github.com/ThiinkMG/FAFSA-Timeline-Generator-Tool/settings).
2.  Select **Collaborators** from the left sidebar.
3.  Click **Add people** and enter their GitHub username or email.
4.  Once they accept the invitation, they can push changes directly to the repository (if given the permissions).

## Project Structure
- `index.html`: Main entry point.
- `css/`: Stylesheets.
- `js/`: JavaScript logic.
- `images/`: Static assets.
